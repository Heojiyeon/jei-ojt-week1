## 재능교육 OJT 1주차 과제 - SVG를 활용한 덧셈 게임 ➕🎮

### 목표
- SVG와 Vanilla TypeScript만을 활용해 SPA 형태로 제작
- 재사용성, 유지보수성, 가독성 향상

### 핵심 기능
- 문제 페이지 첫 진입 시 문제 수 만큼 랜덤 값으로 수식 생성
- 옵션 바를 활용해 해당 문제의 답안 선택 및 확인
- 답안의 정/오답 체크를 통해 결과 페이지에서 정답률 및 통과 여부 확인
